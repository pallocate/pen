KFindNodeOperation(server: KServer, node: KNode, routingTable: KRoutingTable, lookupId: KNodeId)

Finds the K closest nodes to a specified identifier The algorithm terminates when it has gotten responses from the K closest nodes it has seen. Nodes that fail to respond are removed from consideration

Parameters

server - Server used for communication

kadNode - The local node making the communication

lookupId - The ID for which to find nodes close to
